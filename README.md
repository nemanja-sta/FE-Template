# FE-Template

1. Check origin:
git remote -v

2. To remove origin use: 
git remote rm origin

3. Check if origin is removed:
git remote -v

4. Add a new origin:
git remote add origin https://github.com/nemanja-sta/REPO-NAME.git

5. Check if origin is added:
git remote -v

6. Push to origin:
git push -u origin main
